垂直水平居中

垂直居中

方法一

适合单行垂直居中，当元素内容是单行的，并且其高度是固定不变的，将“line-height”设置成和“height”高度一样就行了。但这种方法只适合单行文本元素，在多行中是不行的。

方法二

该方法是给容器设定绝对定位（position: absolute;），并且定位高度和（top: 50%;）和margin-top为高度的一半(margin-top: -height/2)。使用这种方法意味着你的元素必须要有一个固定的高度。

HTML

CSS

使用这种方法还可以实现水平居中

HTML

CSS

使用这种方法要注意：水平居中的元素要有明确的大小，既要确定width和height；给元素绝对定位，设置left和top为50%；最后设置元素的margin-top和margin-left为负值，其值分别为元素高度和宽度的一半。

方法三

该方法是用div模拟表格的效果，也就是将多个div的display属性设置为table和table-cell，并设置它们的vertical-align的属性为middle。该方法没有高度限制，可以根据元素内容动态改变高度，也就没有了空间的限制。但这种方法只适合现代浏览器，在IE6-7下无法运行。

CSS

为解决兼容性问题，需要额外增加一个div，并使用hack。

CSS

方法四

该方法是在需要居中的元素前放一个空的div，然后设置这个div的高度为50%，margin-bottom为元素高度的一半，而且居中元素需要清除浮动。使用这种方法，如果居中元素放在body中，需要给html和body设置高度为100%。

CSS

方法五

方法五与方法三一样都是使用table-cell来实现，不过不同之处是在该方法中我们需要一个线盒型，用来实现IE下的效果，需要增加一个行内标签，并把线盒型的高度设置为100%。

水平居中

方法一

使用margin: auto配合元素的width实现。该方法要满足两个条件：一，元素要有一个固定宽度；二，元素的margin-left和margin-right必须设着为auto。

HTML

CSS

方法二

使用绝对定位配合负margin来实现。该方法也需要一个固定宽度，并对元素绝对定位，left值为50%，负margin-left值等于宽度的一半。另外如果元素不是相对浏览器的话，还需要在其父元素上设置position:relative。

HTML

CSS

方法三

该方法主要针对单行文本居中或者前文介绍的table格式居中，用text-align: center实现。
